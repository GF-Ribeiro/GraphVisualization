# GraphVisualization
Tool for graph visualization and graph algorithms showcase.

Developed with Unity 2020.2.2f1

Currently supporting directed and undirected graphs.

Algorithms:
* Kruskal
* Prim
* Depth First Search
* Breadth First Search
* Kosaraju
